Made to lower design boundaries

Galaxy removes the frustration and hacks out of front-end, like positioning elements, styling component layers, and state-toggling.

Markup first

Using the g attribute allows semantic markup without limitations. You'll never have to touch CSS again.

Natively fast

With Galaxy.js, components and their state animations perform purely on CSS.

Galaxy Grid

Since Galaxy Grid was made with Flexbox, building layouts are really easy and hack free.

A production ready framework

In addition to a 2kb Javascript footprint, Galaxy's robust markup and design is the web's missing framework.
